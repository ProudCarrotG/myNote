# 数值计算
> 记忆技巧：
> 1. P: 代表除号
> 

1. CPI：每个指令需要多少周期
	1. C：cycles，周期，表示计算机的 CPU 周期
	2. P：per，每
	3. I：instrcition，指令
2. IPS：每秒能执行多少条指令
	1. I：指令
	2. S：时间的单位（秒）
	3. M：百万 $1e6$
3. 题目中给出的某 MHz 处理机的参数表示一秒有多少个 CPU 周期
4. 加速比
	1. $加速比 = {旧执行时间}/{新执行时间}$


# 编码优化
> 优化指令对应的编码


1. 哈夫曼编码
	1. 由使用频度由低到高构建哈夫曼树
	2. 左儿子做 0，右儿子做 1
	3. 每往下一个深度，编码长度+1
	4. $平均码长=Σ(该指令的占比*该指令的码长)$
2. X-X 等长扩展码
	1. 限定几个长度的扩展码
	2. 保证高使用频率的操作码短
	3. 保证每个指令都有对应的码
